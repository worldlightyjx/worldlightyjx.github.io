全排列的几种实现方式

发表于 2019-04-02 更新于 2025-02-23 分类于 算法与数据结构

元素不重复

例如：[1,2,3] -> [[123],[132],[213],[231],[321],[312]]

  • 每一位实验，如果之前没有用过就加入，时间复杂度为O(n! * n)
  • 利用元素之间交换的思想(就不需要查重了)

元素有重复

例如：[1,1,2] -> [[112],[121],[211]]:

  • 借助used数组去重
  • 使用交换方式
